STAGING-- when we add the file it goes to staging and only those file we add which we want them to go to commit[commit- is a snapshot of file] .here we add file which goes to staging and then we commit them in git which let this file to go to unmodified stage.
UNMODIFIED--when file reach to this stage means files are saved in git. here if you want to remove the file then you can remove it.
MODIFIED-- at unmodified stage,you can edit the file which lead it to modified stage and once it reach modified stage ,we have to stage the file and commit the file to save the changes made in file.
